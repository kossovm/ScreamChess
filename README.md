Scream Chess 🧠♟️
🔗 Live Demo: [https://b2a.kz/ScreamChees]

🎯 Зачем я это сделал?
Онлайн-шахматы стали стерильными. Классические платформы анализируют только ходы на доске (логику), но полностью игнорируют то, как человек мыслит и ломается под давлением (эмоции).

Я захотел вернуть в игру человеческий фактор и создал PsychoVoice — гибрид соревновательного GameFi, HR-Tech и AI-коучинга. Это платформа, где шахматная партия становится окном во внутренний мир игрока.

🚀 Уровень «Великий»: Что под капотом?
1. Голосовой хаос (Vocal Warfare & Live Chat)
Шахматы — это битва умов. Я интегрировал WebRTC, чтобы реализовать открытый войс-чат в мультиплеере. Вывести соперника из равновесия, использовать трэш-ток — это часть геймплея.
Управлять игрой можно без мыши, используя Web Speech API. Просто диктуй: "e2 e4", "knight to f3", "длинная рокировка", "сдаюсь" (поддержка RU/EN).

2. Psycho-AI Профайлинг (Heuristics + Gemini)
Мой алгоритм (src/lib/psychoAnalyzer.ts) собирает метрики в реальном времени:

Толерантность к риску: Частота взятий, шахов и зевков.

Импульсивность: Процент ходов, сделанных быстрее 2.5 секунд.

Терпение и Стресс: Процент долгих раздумий (15s+) и дрожание/активность голоса.

После игры эти сырые данные летят в Gemini API (/api/psycho), который выдает жесткий, честный «Психологический паспорт» пользователя: его когнитивный стиль и лидерские качества.

3. Полноценный Игровой Движок и Мультиплеер
Правила ФИДЕ: Полная валидация через chess.js (взятие на проходе, рокировки, превращения).

AI Оппонент: Интеграция Stockfish (11 уровней сложности) через WASM в Web Worker, чтобы не блочить UI.

Онлайн PvP: Игра по ссылке. Синхронизация ходов через Supabase Realtime.

Локальный PvP: Для быстрой игры на одном экране.

4. Социальный слой и B2B потенциал
Все данные сохраняются в облаке (Supabase). Я сделал глобальные и гео-лидерборды по городам.
Продукт готов к монетизации: POST /api/psycho — это готовое B2B HR-Tech решение. Рекрутер дает кандидату сыграть 5-минутный блиц и мгновенно получает срез его риск-профиля.

🛠 Технический Стек
Фронт: Next.js 14 (App Router), React 18, TypeScript, Tailwind CSS, Framer Motion.

Логика: chess.js + react-chessboard + Stockfish.js (WASM).

Голос и Связь: Web Speech API (распознавание), WebRTC (войс-чат), MediaStream.

Бекенд и База: Supabase (Auth + Postgres + Realtime).

Мозг (ИИ): Gemini API (анализ психотипа).
